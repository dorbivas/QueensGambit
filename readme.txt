Knight's tour puzzle 
Idan Oren & Dor Bivas 

Refrence to the thoery:
https://en.wikipedia.org/wiki/Knight%27s_tour

The project devided to 7 parts:
Q1 - generting knights matrix of possiable moves
Q2 - translating the matrix to a single linked list 
Q3 - generating a tree of all path (based on all linked list in Q2)
Q4 - save the complete path
Q5 - binery compressing and writing the path to file
Q6 - binery reading and generating a complete path from file.
Util - utulty functaions
